# Tóm tắt

Gọi $p, q$ là hai trạng thái của 2 game khác nhau, $N(p)$ là tập các trạng thái đến được từ p.

$$SG(s) = Mex (SG(N(s)))$$
$$SG\left((p, q)\right) = SG(p) \oplus \ SG(q)$$
